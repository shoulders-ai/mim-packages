# References

Reference library for capturing and citing sources from a real, verifiable collection.

Add references by DOI, PDF, or pasted citation. The library backs the editor's citation autocomplete and export bibliography. The AI can search, retrieve, and cite from the library so every reference in your documents is grounded in an actual source.

**Tools:** `references.search`, `references.get`, `references.add`, `references.import`, `references.attach_pdf`
